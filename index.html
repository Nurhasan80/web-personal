Tentang Saya

Nama asli saya Nurhasan biasa dipanggil Bang Achan. Saya adalah pemilik dari web potfolio ini.

Saya mulai mengenal internet sejak masuk kuliah tahun 2003. Pada waktu itu saya hanya menggunakan internet untuk sekedar tugas-tugas kuliah, googling, dan download musik.

Sejak masa kuliah itulah saya mulai mendalami ilmu komputer khususnya dunia pemrograman karena kebetulan saya mengambil jurusan Manajemen Informatika dan Komputer. Dari sinilah awal mula saya tertarik dengan coding.

Sekitar tahun 2006 saya mulai mengenal dan belajar membangun sebuah blog. Tapi karena saya termasuk orang yang tidak suka menulis, saya lebih banyak menghabiskan waktu untuk edit-edit template blog ketimbang menulis artikel.

Karena waktu itu saya kuliah sambil bekerja saya ingin memiliki uang tambahan dari internet salah satunya adalah membuat blog Adsense dan mengikuti survey-survey online.

Singkat cerita karena penghasilan blog Adsense tidak menentu kemudian pada tahun 2016 saya mengenal namanya crypto currency terutama Bitcoin, disinilah saya mulai berinvestasi di Bitcoin sampai sekarang.
